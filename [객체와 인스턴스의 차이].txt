[객체와 인스턴스의 차이]

1. 객체 (Object)
객체지향프로그래밍(OOP)에서 생성된 구체적인 데이터 형태
클래스를 기반으로 실체화된 것으로 해당 클래스로부터 생성된 인스턴스를 의미
객체는 클래스의 인스턴스로서, 클래스에 정의된 속성과 동작을 구체화한 것
-> 클래스로부터 생성된 인스턴스는 객체라고 부를 수 있다.

2. 인스턴스 (Instance)
객체의 실체화된 형태
클래스를 기반으로 생성된 객체를 가리킴
생성자를 통해 객체를 실체화하면 해당 객체는 인스턴스가 된다.
-> 객체와 같은 의미를 지니지만, 구체적인 표현으로 해당 클래스로부터 실제로 생성된 것임을 강조

=> 자바에서 해당 클래스로 부터 생성된 것이 객체이고  객체의 실체화된 형태가 인스턴스이다.
