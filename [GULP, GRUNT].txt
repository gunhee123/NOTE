[gulp] (자동화 빌드 도구 / JS 기반의 빌드 시스템 / Node.js 환경에서 실행 됨)

1. 파일 압축과 병합
2. SCSS/LESS 등 CSS 전처리기 지원
3. 자동 새로고침 (라이브 리로딩)
4. 이미지 최적화
5. 코드 문법 검사 및 오류 찾기
6. 웹 개발 작업의 자동화

-> 개발과정을 자동화하고 웹 개발 작업을 효율적으로 처리함

'gulpfile.js' 설정 파일을 작성하여 정의
Gulp는 다른 빌드 도구인 Grunt와 함께 널리 사용 됨. 효율적인 웹 프로젝트를 관리할 수 있도록 도와줌

(gulp : 스트리밍 방식)

***

[grunt] (gulp와 같이 JS 기반의 빌드 시스템)


(grunt : Task 기반의 방식 - 설정 파일에 명시적으로 세부적인 작업을 정의하며 사용)
