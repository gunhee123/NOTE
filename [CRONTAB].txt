[crontab]
Unix 계열 운영체제에서 주기적인 작업을 예약하기 위해 사용되는 명령어와 설정파일이다.

cron은 시간 기반 스케줄러로 특정 시간에 스크립트 또는 명령어를 실행할 수 있도록 해줌.

주로 백업, 로그 파일 정리, 데이터베이스 백업 등의 작업에 사용됨. 특히 서버 관리 및 자동화에 유용.

<crobtab 사용>
1. crontab 파일 설정 : 'crontab' 명령을 통해 자신의 crontab 파일을 편집하거나 생성 가능.
	(주기적으로 실행되길 원하는 명령어들을 포함하고 있다.)

2. crontab 작성 규칙 : 일반적으로 분, 시, 일, 월, 요일 등의 시간 정보와 실행할 명령어 입력

3. 주기적 작업 예약 : 작성된 명령어들은 설정된 시간에 주기적으로 실행됨. 주기는 매 분, 매시간, 매일, 특정 요일 등으로 설정 가능

ex) 매일 오전 3시에 데이터베이스 백업 스크립트를 실행하려면 crontab 파일에 해당 작업을 등록할 수 있다.


* 스케줄러 : 작업을 예약하고 실행하는 시스템 또는 도구
